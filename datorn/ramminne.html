Vad är ett RAM-minne?

RAM eller RAM-minnet är ett minne som lagrar data temporärt som din dator behöver snabb tillgång till. Ett RAM hjälper ditt system att fungera smidigt av att låta CPU:n att hämta data från SSD:s eller HDD:s snabbare. RAM instabil, alltså tappar den all sin data när du stänger av datorn.

Hur mycket RAM ska man ha?

Mängden RAM räknas i gigabytes (GB), vilket avgör hur många “uppgifter” din dator kan hantera på en och samma gång. Om du använder din dator för småsaker så fungerar 8GB RAM helt bra, fast om du ska behöva göra jobb med högre krav som till exempel att spela datorspel så borde du sikta på 16GB RAM. Hastigheten som ditt RAM jobbar med är kompatibelt med ditt moderkort, CPU och den typ som kan vara DDR4 eller DDR5 som är viktigt att kolla när man väljer rätt RAM för ditt system.
